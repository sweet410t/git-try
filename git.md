```
    Workspace：工作区
    Index / Stage：暂存区
    Repository：仓库区（或本地仓库）
    Remote：远程仓库
```
###常用6大命令
1. add

 
   ```
        git add *
        git add <filename>
    ```
2. commit


    ```
    git commit -m "代码提交信息"
    ```
3. push
4. fetch/clone
5. checkout
6. pull
